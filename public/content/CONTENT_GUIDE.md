## Content Creation Guidelines

### For All Visual Content:

**Image Requirements:**
- Resolution: Minimum 1920x1080px for clarity
- Format: PNG for illustrations, JPG for photographs
- Color Palette: High contrast with distinct colors
- Text: Minimum 24px, bold, sans-serif fonts
- Labels: Clear, concise, positioned near relevant parts
- Arrows: Thick, colored, showing direction of processes

**Animation Requirements:**
- Duration: 60-120 seconds per topic
- Format: HTML5 with CSS animations (browser-compatible)
- Silent: NO audio files, all information visual
- Loop: Auto-repeat for reinforcement
- Speed: Slow enough for comprehension (not too fast)
- Transitions: Smooth, clear visual flow

**Accessibility Standards:**
- Can be understood without hearing
- Text provides complete information
- Visual metaphors are clear
- Process sequences numbered
- Color is not the only differentiator (use shapes/labels too)
